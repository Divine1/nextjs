# Nextjs Template

## setting up the project

`npm install`

## Development mode

`npm run dev`

## Production mode 

`npm run build`
`npm run start`

- optimized files will be generated into assets directory

## Generate static pages

`npm run build`
`npm run export`

- static files will be exported into out directory

## Scss config

- scss is configured in `assets\scss\style.scss`
